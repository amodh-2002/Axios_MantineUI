# Axios and Mantine UI Project

## Quick Start

1. Clone the repository:

   ```bash
   git clone https://github.com/amodh-2002/Axios_MantineUI.git
   cd Axios_MantineUI
   ```

   ```
   npm install
   yarn install
   ```

   ```
   npm run dev
   OR
   yarn dev
   ```
2. DummyData Used:
   https://dummyjson.com/docs/products
